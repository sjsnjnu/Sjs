    # 介绍VUE

    vue是一套用于构建用户界面的渐进式框架。与其它大型框架不同的是，Vue 被设计为可以自底向上逐层应用。Vue 的核心库只关注视图层，不仅易于上手，还便于与第三方库或既有项目整合。🥲

    “它是一个可以与您一起成长并适应您的需求的框架”🧐

    渐进式：灵活？VUE可以按照不同方式使用···

    # 单文件组件

    用类似于HTML的文件格式创建Vue组件，称为SFC， HTML和CSS封装在一个文件中

    # SFC语法规范

    Vue文件由三种类型的顶层语法块组成：<template>、<script>、<style>

    • <template>

      每个*.vue文件只能有一个顶层<template>块

      其中的内容会被提取出来并传递给@vue/conpiler-dom，预编译为JavaScript的渲染函数，并附属到导出的组件上作为其render选项

    • <script>

      每个*.vue文件只能有一个顶层<srript>块

      该脚本将作为ES Module来执行

      其默认导出的内容应该是Vue组件选项对象，它要么是一个普通的对象，要么是defineComponent的返回值

    • <style>

      可以有多个<style>标签🏷️

    # Quick Start
