    # 子网掩码

    子网掩码 ➡️ 网络号

    why：

    方便管理（为了通信，需要广播）

    广播不会经过路由器（物理分离）

    计算机在不同的子网中，数据会被发送到默认网关，也就是路由器，然后路由器再发送

    • A 1.0.0.1 -- 127.255.255.254

    • B 128.0.0.1 -- 191.255.255.254

    • C 192.0.0.1 -- 223.255.255.254

      类别 最大网络数 IP地址范围
      A 126(2^7 - 2) 1.0.0.1 - 127.255.255.254
      B 16384(2^14) 128.0.0.1 - 191.255.255.254
      C 2097152(2^21) 192.0.0.1 - 223.255.255.254

      CIDR表示

      # Wi-Fi VS Bluetooth

      无线连接电子设备的射频技术

      # Default Gateway

      一般来说是路由器，路由器是一个网络的网关或者入口

      在同一个网络上的计算机，可以通过交换机通信

      每个子网都有自己的默认网关

      如果在同一网络，直接广播ARP得到MAC地址，然后就能通信

      先检查IP，发现不是同一网络，发出一个ARP广播，询问默认网关的MAC（ARP不能经过路由器）

      # APIPA

      Automatic Private IP Addressing（自动私有IP寻址）

      如果DHCP服务器故障，计算机本身将自行分配自己的Ip地址，范围是169.254.0.1 -- 169.254.255.254，这样就能仍然和同一本地网络上的设备通信

      还有一个16位的子网掩码255.255.0.0

      # Monitor

      1. CRT：Cathode Ray Tube 3色电子枪 refresh rate： 60 Hz 每秒60次重绘

      2. LCD ： Liquid Crystal Display 灯（荧光灯，LED）➡️液晶➡️显示

        LED功耗更低且具有higher contrast ratio（对比度），更黑和更白1000:1代表白色是黑色的1000倍亮

        LCD有三种不同的面板：TN、VA、IPS

        TN：twisted nematic(扭曲向列)具有最高的刷新率和响应时间，色彩重现不行 视角不行

        IPS：in- plane switching 更好更准确的色彩重现

        VA：Vertical alignment 响应时间介于二者中间 色彩也是

      3. OLED : organic light-emitting diode

        和LCD最大的不同是光源，每个像素都会产生单独的照明，不需要单独的背光，电流施加到有机化合物上，色彩更出色，对比度更好 使他们发光 对比度100000：1

      4. 分辨率： 1280*720（HD） 1920*1080（FHD）2560*1440(QHD、2K) 3840*2160(4K)

      # Wireless Bridge Mode

      无线桥接模式是内置于Wi-Fi路由器中的一项功能，将两个网络无线连接成一个 ，如果一个路由器设为桥接模式，那么他就直接接入到另一个网络中，合并成一个，并且不具备防火墙、NA T或者分配IP的工作

    桥接模式同样

    # LAN、WAN、SUBNET

    • LAN： local Area Network 局域网（专用网络）

    • WAN： Wide Area Network 广域网

    • SUBNET：subnet work 子网

    # public vs private IP ADDRESS

    • PUBLIC : 在互联网上注册的、唯一的

    • PEIVATE：IPv4短缺造成，当需要上网时，通过NAT（network address translation）获得一个公共IP地址

    查看私有IP：ipconfig

    查看公共IP：百度输入IP

    # MAC ADDRESS

    由一个6字节的十六进制数字组成

    IP：local 定位

    Mac: identify 识别

    输入google.com会发生什么：

    DNS将google.com 改成IP 不在同一个网络 转发到默认网关，即路由器， ARP广播 路由器的mac 路由器拥有数据，将检查IP 确定最佳路径， 转发到下一个路由器 最终的路由器需要google服务器的mac地址

    # Wi-Fi6

    Wi-Fi6的主要目标不仅是提高速度，而是在大量设备连接到Wi-Fi网络时，使Wi-Fi网络性能更好

    将互联网带宽分解为单独的流并将其推送到连接的设备

    # Cloud Computing

    • IaaS（Infrastructure as a Service）基础设施即服务：个人负责应用程序，数据，操作系统，中间件，运行时，如在线数据备份
    • PAAS（Platform as a Service） 平台即服务： 个人只负责应用程序和数据
    • SaaS（Software as a Service） 软件即服务：：最常见。如Google dos

    # ifconfig

    显示网络适配器的TCP/IP网络配置

    lo0: Loopback(环回地址)，loopback接口是一种纯软件性质的虚拟接口，接口创建后，物理层和链路层永远处于up状态。在IPv4中，永远为127.0.0.1，即标准域名localhost
# TCP/IP协议栈中的loopback接口
1. 用于网络服务测试，避免由于网络接入带来的安全问题：
一般用作client/server类的网络服务的测试，client与server运行在同一台主机上，client通过使用loopback地址访问server，如在web服务中，用http://127.0.0.1或http://localhost/来访问本地的server服务
2. 测试IP协议栈
通过ping loopback地址的方式测试操作系统中IP协议栈是否正常
3. 在网络中，所有源地址属于loopback地址的数据包将会被丢弃
# 网络设备中的loopback
没有”回环“的意思，此虚拟接口被分配到一个IP地址，没有对应的实际的物理接口，主要用于管理目的，例如设备发出的报警。也可以理解这种地址是网络设备提供的某个服务的地址
# 通信中的loopback
指通过软件或硬件的方式，将接收到的信号或数据直接返回给发送者的测试方法，“回环测试”
# 硬件loopback
物理上把发送和接收通道相连，测试不同位置两点之间的电路连接

ping 172.0.0.1或者ping localhost 如果操作成功，则网卡正常
